# Music Recommendation System using Facial Emotion Recognition

## Overview
This project is an AI-based music recommendation system that suggests songs according to the user’s facial emotions. The system detects facial expressions in real time using computer vision and machine learning techniques, then recommends suitable music based on the detected mood.

The project aims to enhance user experience by creating a personalized and emotion-aware music recommendation platform.

---

## Features
- Real-time facial emotion detection
- Mood-based music recommendation
- Face detection using OpenCV
- Emotion classification using deep learning
- Interactive and user-friendly system
- Personalized music suggestions

---

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- Pandas
- Deep Learning
- Machine Learning

---

## Emotions Detected
The system can identify emotions such as:
- Happy
- Sad
- Angry
- Neutral
- Surprise

Based on the detected emotion, appropriate songs are recommended to the user.

---

## Project Workflow
1. Capture user face using webcam
2. Detect facial expressions
3. Classify user emotion
4. Match emotion with music category
5. Recommend suitable songs
